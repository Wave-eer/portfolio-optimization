# Time Series Forecasting for Portfolio Management Optimization

This project applies time series forecasting to historical financial data to enhance portfolio management strategies at GMF Investments.

## Project Structure

├── .github/

├── .gitignore
├── requirements.txt
├── README.md
├── data/

├── notebooks/
│   ├── __init__.py
│   ├── README.md

## How to Run

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run Notebooks**:
   - Open Jupyter Lab or VS Code and run notebooks in the `notebooks/` directory sequentially.
